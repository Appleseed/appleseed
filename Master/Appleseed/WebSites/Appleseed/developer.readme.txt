Before compiling, please copy and rename the web.config.standard file to web.config.
 This allows you to have a personal copy of web.config without uploading it to the server.

VS 2015 is recommended to develop Appleseed Portal

Simply open Appleseed.sln for our project.

Make sure you have a SQL Database configured

Control f5 

The software will take you to /Installer and walk you through install
